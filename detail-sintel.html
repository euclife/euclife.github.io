SINTEL LAA

Application intended for every employee of the Sintelis division at PT Kereta Api Indonesia. This apps allows you to set an appropriate template to dynamically load in a service record form when a given category is selected. It can be used as-is to survey the state of signalling, telecommunication and electricity infrastructures. The app contains fields for documentation of the generated dynamic infrastructure checklist form, surveyor info, date, and more. Additionally, the app can be used to collect photos and information for documenting distress. As with every app in this gallery, this mobile inspection form can easily be modified to include additional information and works offline in case of low or no internet signal. Features developed for this project are authentication with HRIS, user management, checklist management, synchronize with SAP, dynamic form, dynamic report, notification for mobile with One Signal, Notification CMS with Pusher, Apps Debugging with Laravel Telescope, Report Management.

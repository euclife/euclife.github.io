EDU KAI

Learning application designated for every PT Kereta Api Indonesia employees. This application allows to learn various kinds of curriculum to train new employees at PT Kereta Api. You can take lessons starting from the basics, watching videos, to online tests. And allows you to see the achievements that you have made. Trainers can also add new curriculum or lessons that can later be learned by new employees. Features developed for this project are powerpoint player, pdf player, curriculum management, test management, score management, notification for all employee.
